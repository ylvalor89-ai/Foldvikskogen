# Foldviksskogen – Innredningsplanlegger

En liten, personlig nettside for å planlegge innredningen av huset på Foldviksskogen.

## Hva den gjør
- Meny med alle rommene i huset (Barnerom, Kontor/lager, Ylvas soverom, Amunds soverom, Stue, Kjøkken, Bad, Gang, Uteplass, Veranda)
- Legg inn inspirasjonsbilder, moodboard og AI-bilder per rom
- Legg inn produkter med bilde, pris, kjøpelenker og status (Vurderer / Ønsker / Bestilt / Kjøpt)
- Fargepalett, mål og notater per rom
- Oversiktsside med samlet budsjett og status
- Alt lagres lokalt i nettleseren – eksporter/importer for backup

## Hvordan bruke
Åpne `index.html` i nettleseren, eller besøk den publiserte siden (GitHub Pages).

Dataene dine lagres i din egen nettleser. Bruk **Eksporter** for å ta backup, og **Importer** for å flytte dataene til en annen maskin eller nettleser.
